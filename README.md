# Streams opencv

## Run Program
Make a .env file in main folder to configure. Add macros.

program:
- singlecapture.py
- savevideo_tofile.py
- test_stream.py

```
python3 program
```
## Configure and macros for .env file

### singlecapture.py
```
SENSORSTXT_PATH 
```
### savevideo_tofile.py
```
TIME_TO_SEND
SENSORSTXT_PATH 
FOLDER_TOSAVE
ENABLE_GUI
SAVE_VIDEO
```
### test_stream.py
```
ENABLE_GU
SENSORSTXT_PATH
```
